## Spécifications techniques

### Desktop
- **Dimensions recommandées:** 1920x1080px ou 2560x1440px
- **Format:** JPG ou WebP
- **Poids:** < 500KB (optimisé pour web)
- **Couleurs principales:** #4dafef (primary), #ffffff (white)
- **Style:** Minimaliste, professionnel, moderne

### Mobile
- **Dimensions recommandées:** 750x1334px (iPhone) ou 1080x1920px (Android)
- **Format:** JPG ou WebP
- **Poids:** < 300KB (optimisé pour mobile)
- **Couleurs principales:** #4dafef (primary), #ffffff (white)
- **Style:** Minimaliste, professionnel, moderne, optimisé vertical

---

## Notes importantes

1. Les images doivent être suffisamment subtiles pour ne pas distraire du contenu texte
2. Les couleurs doivent correspondre à la palette du site (primary: #4dafef, secondary: #e54a33, tertiary: #1e3a5f)
3. Le style doit être cohérent avec l'identité visuelle de Laundry.ma
4. Les images doivent être optimisées pour le web (compression appropriée)
5. Pas de texte, pas de personnes, juste des éléments de design abstraits
